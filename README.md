## Description

Welcome to the **AC Manager** app! This application has been designed for a company specializing in air conditioning services. The app allows managing service orders, scheduling installations, and tracking customer history. With this app, the company can enhance its customer service efficiency and optimize its team's operations.

# Installation Instructions

1. **Clone the source code:**

git clone <repository-url>
cd <project-name>

2. **Install the required dependencies using npm or yarn:**

npm install

1. **System-specific requirements:**

- **iOS (macOS):** 
- If you plan to run the app on an iOS simulator or physical iOS device, make sure you have Xcode installed with the latest Command Line Tools. You can download Xcode from the Mac App Store.

- **Android (Windows/macOS/Linux):** 
- If you plan to run the app on an Android emulator or physical Android device, ensure you have Android Studio installed with the necessary components for the Android version you intend to test. You can download Android Studio from the official website.

5. **Start the app in development mode:**

npm run start

6. **Scan the displayed QR code using the Expo Go app on your mobile device or run the app on an Android/iOS simulator or physical device using the appropriate options in the Expo Developer Tools.**
